KEA PDF to Excel Reorder Tool

Install:
1. Install Python 3.10 or above.
2. Open Command Prompt in this folder.
3. Run: pip install -r requirements.txt
4. Run: streamlit run app.py

Use:
1. Upload KEA option report PDF.
2. Edit New Option No in the app or download Excel.
3. Copy Console Lines.
4. Paste into the master console script prompt on KEA page.
5. Verify manually and click Update Options.

Note:
Use 0 only for deleting an option. Always download final option report and verify.
